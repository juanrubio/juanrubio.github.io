# juanrubio.github.io

Personal website and blog for Juan A. Rubio.

## Stack

This site is built with Jekyll and based on the Beautiful Jekyll theme, with
local customizations to content, layout, and styling.

## Local development

If you have Ruby and Bundler available:

```bash
bundle install
bundle exec jekyll serve
```

Then open:

```text
http://127.0.0.1:4000
```

## Structure

- `_posts/` — blog posts
- `_layouts/` — page and post templates
- `_includes/` — shared partials
- `css/` — site styling
- `img/` — images and assets
- `_site/` — generated output

## Notes

- Keep content and copy current, calm, and low-maintenance.
- Prefer small iterative refreshes over full theme rewrites.
- This repo contains both content and theme customizations.
